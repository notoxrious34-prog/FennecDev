# Operator Quick Reference — Fennec Facturation v3.0 ERP
## For Business Operators (Non-Technical)

---

## Daily Operations

### Starting the Application
Double-click the **Fennec Facturation** shortcut on the Desktop.
The application opens to the Dashboard.

### Creating an Invoice
1. Click **Invoices** in the left sidebar.
2. Click **New Invoice** (top right button).
3. Select the customer from the dropdown.
4. Add products by clicking **Add Line Item**.
5. Verify the total amount.
6. Click **Save Invoice**.

### Changing the Language
Click the language switcher in the top navigation bar.
Select: **EN** (English), **FR** (Français), or **عر** (العربية).
The application updates immediately — no restart required.

## Data Backup

### Check Backup Status
1. Click **Settings** (gear icon, bottom of sidebar).
2. Click the **Data Backup** tab.
3. The page shows when the last backup was created and how many backups are stored.

### Create a Manual Backup
On the Backup page, click **Backup Now**.
A green notification confirms the backup was created.

### How Long Are Backups Kept?
The last **7 backups** are kept automatically.
Older backups are deleted automatically to save disk space.

## License

### Where Is My Machine ID?
If the application shows the license activation screen:
Your Machine ID is displayed on the activation screen.
Contact your software provider with this Machine ID to receive your license key.

### Activating a License
1. Copy the license key you received from your software provider.
2. Paste it into the license key input field.
3. Click **Activate**.
4. If successful, the main application opens immediately.

### License Stops Working
If you see "License Expired" or "Machine Mismatch":
Contact your software provider with your current Machine ID (shown on the license screen).
